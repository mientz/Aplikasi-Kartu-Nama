Pengaturan Server Email SMTP

Pengaturan ini mengatur koneksi dengan server email.

  • SMTP Server : Alamat Server Email, bisa berupa Domain ataupun IP Address

  • SMTP Server : Tambahkan Jika memiliki lebih dari satu server email, berguna sebagai backup ketika server utama tidak ada respon

  • SMTP Port : Port Server SMTP

  • SMTP Username : Username akses server SMTP

  • SMTP Username : Password akses server SMTP

  • Email Pengirim : Alamat email sebagai pengirim pesan Contoh : no-reply@example.com maka penerima akan menerima email yang berasal dari no-reply@example.com

  • Nama Pengirim : Nama email sebagai pengirim pesan Contoh : NO-REPLY maka penerima akan menerima email yang berasal dari no-reply@example.com dengan nama no-reply@example.com
